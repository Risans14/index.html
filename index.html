Label Encoding untuk Variabel Kategorikal

# Import label encoder
from sklearn import preprocessing

# label_encoder object knows how to understand word labels
label_encoder = preprocessing.LabelEncoder()
obj = (data.dtypes == 'object')
for col in list(obj[obj].index):
  data[col] = label_encoder.fit_transform(data[col])

Penjelasan:

  • preprocessing.LabelEncoder(): Membuat objek encoder untuk mengubah string ke numerik
  • label_encoder.fit_transform(): Melakukan fitting dan transformasi sekaligus
  • Mengubah nilai kategorikal seperti "Male"/"Female" menjadi 0/1
  • Diperlukan karena algoritma machine learning hanya bekerja dengan angka
  • Contoh transformasi: "Male" ‚Üí 1, "Female" ‚Üí 0, "Graduate" ‚Üí 1, "Not Graduate" ‚Üí 0

Split Data untuk Training dan Testing

from sklearn.model_selection import train_test_split

X = data.drop(['Loan_Status'], axis=1)
Y = data['Loan_Status']
X.shape, Y.shape

X_train, X_test, Y_train, Y_test = train_test_split(X, Y,
                              test_size=0.4,
                              random_state=1)
X_train.shape, X_test.shape, Y_train.shape, Y_test.shape

Penjelasan:

  • X = data.drop(['Loan_Status'], axis=1): Features (semua kolom kecuali target)
  • Y = data['Loan_Status']: Target variable (yang akan diprediksi)
  • train_test_split(): Membagi data menjadi training set dan testing set
  • test_size=0.4: 40% data untuk testing, 60% untuk training
  • random_state=1: Seed untuk reproducibility (hasil yang konsisten)
  • .shape: Menampilkan dimensi data (jumlah baris, jumlah kolom)

Training Multiple Machine Learning Models

from sklearn.neighbors import KNeighborsClassifier
from sklearn.ensemble import RandomForestClassifier
from sklearn.svm import SVC
from sklearn.linear_model import LogisticRegression

from sklearn import metrics

knn = KNeighborsClassifier(n_neighbors=3)
rfc = RandomForestClassifier(n_estimators=7,
                        criterion='entropy',
                        random_state=7)
svc = SVC()
lc = LogisticRegression()

# making predictions on the training set
for clf in (rfc, knn, svc, lc):
  clf.fit(X_train, Y_train)
  Y_pred = clf.predict(X_train)
  print("Accuracy score of ",
      clf.__class__.__name__,
      "=", 100*metrics.accuracy_score(Y_train, Y_pred))

Penjelasan:

  • 4 Algoritma Machine Learning:
    1. KNeighborsClassifier: K-Nearest Neighbors (n_neighbors=3)
    2. RandomForestClassifier: Random Forest (7 trees, entropy criterion)
    3. SVC: Support Vector Classifier
    4. LogisticRegression: Logistic Regression
  • clf.fit(X_train, Y_train): Melatih model dengan data training
  • clf.predict(X_train): Memprediksi data training itu sendiri
  • metrics.accuracy_score(): Menghitung akurasi (persentase prediksi benar)
  • clf.__class__.__name__: Mengambil nama class model
  • Testing pada data training menunjukkan performa model (biasanya tinggi)
Output:
Accuracy score of RandomForestClassifier = 98.64%
Accuracy score of KNeighborsClassifier = 82.07%
Accuracy score of SVC = 69.84%
Accuracy score of LogisticRegression = 80.98%

Evaluasi Model pada Testing Set

# making predictions on the testing set
for clf in (rfc, knn, svc, lc):
  clf.fit(X_train, Y_train)
  Y_pred = clf.predict(X_test)
  print("Accuracy score of ",
      clf.__class__.__name__, "=",
      100*metrics.accuracy_score(Y_test, Y_pred))

Penjelasan:

  • Kode ini sama dengan baris 13, tetapi memprediksi pada data testing
  • Data testing adalah data yang belum pernah dilihat model selama training
  • Akurasi pada testing set menunjukkan kemampuan generalisasi model
  • Perbedaan besar antara akurasi training dan testing menunjukkan overfitting
  • Random Forest biasanya memberikan performa terbaik untuk dataset seperti ini
Output:
Accuracy score of RandomForestClassifier = 78.05%
Accuracy score of KNeighborsClassifier = 63.41%
Accuracy score of SVC = 69.51%
Accuracy score of LogisticRegression = 80.49%

Analisis Hasil:

  • Random Forest: Akurasi training 98.64% ‚Üí testing 78.05% (overfitting)
  • K-Nearest Neighbors: 82.07% ‚Üí 63.41% (overfitting signifikan)
  • SVC: 69.84% ‚Üí 69.51% (stabil, sedikit underfitting)
  • Logistic Regression: 80.98% ‚Üí 80.49% (paling stabil dan akurat)

Kesimpulan: Logistic Regression memberikan performa terbaik dengan akurasi 80.49% dan perbedaan training-testing yang kecil.
